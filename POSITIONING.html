1. CSS Positioning

CSS positioning allows you to control where elements appear on your web page. Here are the main types:

  • Static: Default position, follows normal flow.
  • Relative: Moves relative to its normal position.
  • Absolute: Positioned relative to the nearest positioned ancestor.
  • Fixed: Stays in the same place even when you scroll.
